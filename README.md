# CSV2Mongo

This project demonstrates a simple workflow to merge multiple CSV files, clean column names, and save the resulting dataset to MongoDB using Python.

## Features
- Load CSV files into pandas DataFrames
- Clean and standardize column names
- Identify shared columns between datasets
- Merge datasets based on shared columns
- Store merged data in MongoDB

## Requirements
- Python 3.x
- pandas
- pymongo
- MongoDB running locally or remotely

## Installation
1. Clone the repository:
```bash
git clone https://github.com/yourusername/CSV2Mongo.git
````

2. Install dependencies:

```bash
pip install pandas pymongo
```

3. Run the notebook or Python script.

## Usage

Update the file paths and MongoDB connection details in the script/notebook, then execute to merge your CSVs and save to MongoDB.
